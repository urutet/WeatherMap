#  WEATHERMAP

### App shows weather based on the location chosen with a pointer.

#### Built using
    - [OpenWeatherMap API](https://openweathermap.org/)
    - [SnapKit](https://github.com/SnapKit/SnapKit)
    - [Alamofire](https://github.com/Alamofire/Alamofire)
